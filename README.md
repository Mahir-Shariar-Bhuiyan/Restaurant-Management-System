# Restaurant Management System 🍽️

A Python-based Restaurant Management System built using Object-Oriented Programming (OOP).

---

## 📌 Project Overview
This project simulates a real-world restaurant management system with both **Admin** and **Customer** functionalities.

---

## ✨ Features
- 👨‍💼 Admin & Customer management system  
- 🍔 Food item / menu management  
- 📦 Cart system  
- 🧾 Order handling system  
- 👥 Employee management  
- ⚖️ Quantity checking system  
- 💰 Billing system  

---

## 🛠️ Tech Stack
- Python 🐍  
- Object-Oriented Programming (OOP)
